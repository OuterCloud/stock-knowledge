## 一、什么是 RSI？

**RSI（Relative Strength Index，相对强弱指数）** 是由技术分析大师 **J. Welles Wilder Jr.** 在 1978 年出版的《技术交易系统新思路》（_New Concepts in Technical Trading Systems_）中提出的**动量型振荡指标**。

- **作用**：衡量价格变动的速度与幅度，判断市场是否处于**超买（Overbought）或超卖（Oversold）** 状态，并识别潜在的**趋势反转信号**。
- **取值范围**：0 到 100。
- **默认周期**：14（Wilder 推荐），但可根据交易风格调整。

---

## 二、RSI 的数学原理与计算公式

### 1. 基本逻辑

RSI 并非比较“上涨股票 vs 下跌股票”，而是衡量**同一资产在一段时间内上涨力度与下跌力度的比值**。

### 2. 计算步骤（以 N=14 为例）

#### Step 1：计算每日价格变化

对每个交易日：

- 若今日收盘价 > 昨日收盘价 → 上涨幅度 = 今日收盘 - 昨日收盘，下跌幅度 = 0
- 若今日收盘价 < 昨日收盘价 → 下跌幅度 = 昨日收盘 - 今日收盘，上涨幅度 = 0
- 若持平 → 两者均为 0

#### Step 2：计算平均上涨（AU）与平均下跌（AD）

Wilder 使用**平滑移动平均（Smoothed Moving Average）**，而非简单平均：

```math
\text{AU}_t = \frac{(\text{AU}_{t-1} \times 13) + \text{今日上涨}}{14}
```

```math
\text{AD}_t = \frac{(\text{AD}_{t-1} \times 13) + \text{今日下跌}}{14}
```

> 注：前 14 天可用简单平均初始化。

#### Step 3：计算相对强度（RS）与 RSI

```math
RS = \frac{\text{AU}}{\text{AD}}
```

```math
RSI = 100 - \left( \frac{100}{1 + RS} \right)
```

> ✅ 当 AD = 0（连续上涨）→ RS → ∞ → RSI = 100  
> ✅ 当 AU = 0（连续下跌）→ RS = 0 → RSI = 0

## 三、标准解读：超买与超卖

| RSI 区间 | 市场状态 | 含义                                 |
| -------- | -------- | ------------------------------------ |
| **> 70** | 超买     | 价格上涨过快，可能回调（非必然见顶） |
| **< 30** | 超卖     | 价格下跌过猛，可能反弹（非必然见底） |
| **50**   | 中轴线   | 多空分界；突破 50 常视为趋势启动信号 |

> ⚠️ 注意：在**强势牛市**中，RSI 可长期 >70；在**熊市**中可长期 <30。因此，**不能机械照搬 70/30**。

## 四、核心交易信号（从基础到进阶）

### 1. **经典超买/超卖信号**

- RSI > 70 后回落 → 考虑卖出
- RSI < 30 后回升 → 考虑买入  
  ✅ **适用场景**：震荡市、区间行情

### 2. **中心线（50）穿越**

- RSI **上穿 50** → 动量转强，多头占优 → **买入信号**
- RSI **下穿 50** → 动量转弱，空头主导 → **卖出信号**  
  ✅ **优势**：在趋势市中比 70/30 更可靠

### 3. **背离（Divergence）—— 高级预警信号**

| 类型       | 价格走势 | RSI 走势           | 含义               |
| ---------- | -------- | ------------------ | ------------------ |
| **顶背离** | 创新高   | 未创新高（或走低） | 涨势衰竭，警惕反转 |
| **底背离** | 创新低   | 未创新低（或走高） | 跌势放缓，可能反弹 |

> 💡 **成功率最高**的 RSI 信号之一，尤其在日线/周线级别。

### 4. **RSI 形态与支撑阻力**

- RSI 图表中也会形成**头肩顶、双底、三角形**等技术形态
- RSI 的前期高点/低点可作为**阻力/支撑位**
  - 例如：RSI 多次在 60 遇阻回落 → 60 成为阻力

### 5. **失败摆动（Failure Swing）—— 强反转信号**

由 Wilder 提出，比普通超买/超卖更可靠：

- **顶部失败摆动**（看跌）：

  1. RSI > 70（超买）
  2. 回落至 70 以下
  3. 再次反弹但**未突破前高**
  4. 随后跌破前低 → 确认卖出信号

- **底部失败摆动**（看涨）：
  1. RSI < 30（超卖）
  2. 反弹至 30 以上
  3. 再次回落但**未跌破前低**
  4. 随后突破前高 → 确认买入信号

## 五、参数优化：不同周期的 RSI 应用

| 周期          | 特点             | 适用场景                         |
| ------------- | ---------------- | -------------------------------- |
| **N = 2–6**   | 极敏感，信号频繁 | 超短线、日内交易（需过滤假信号） |
| **N = 9–10**  | 敏感度适中       | 波段交易、加密货币               |
| **N = 14**    | 标准平衡         | 股票、期货日线（Wilder 推荐）    |
| **N = 21–28** | 平滑、滞后       | 长线投资、周线趋势确认           |

> ✅ **建议**：
>
> - 短线交易者用 RSI(9) + RSI(14) 双周期验证
> - 长线投资者关注周线 RSI(14) 的背离

## 六、实战组合策略（提升胜率）

### 1. **RSI + 移动平均线（MA）**

- 股价在 200 日均线上方 + RSI > 50 → 只做多
- 股价在 200 日均线下方 + RSI < 50 → 只做空

### 2. **RSI + MACD**

- MACD 金叉 + RSI 从 30 以下回升 → 强买入信号
- MACD 死叉 + RSI 从 70 以上回落 → 强卖出信号

### 3. **RSI + 布林带（Bollinger Bands）**

- 价格触及下轨 + RSI < 30 → 高概率反弹
- 价格触及上轨 + RSI > 70 → 高概率回调

## 七、常见误区与局限性

| 误区              | 正确理解                                        |
| ----------------- | ----------------------------------------------- |
| “RSI >70 就要卖”  | 在牛市中，RSI 可长期 >70，盲目卖出会错过主升浪  |
| “RSI <30 就要买”  | 在熊市中，RSI 可长期 <30，抄底易被套            |
| “RSI 能预测顶底”  | RSI 只反映动量，不预测价格，需结合结构确认      |
| “所有品种都用 14” | 加密货币波动大，可用 RSI(9)；蓝筹股可用 RSI(21) |

> ❗ **最大风险**：在**单边趋势行情**中，RSI 会持续超买/超卖，导致逆势操作亏损。

---

## 八、专业级使用建议

1. **不要单独使用 RSI**：必须结合价格结构（支撑/阻力、K 线形态）、趋势方向、成交量。
2. **优先看背离和失败摆动**：比简单超买/超卖信号可靠得多。
3. **区分市场状态**：
   - 震荡市 → 用 70/30 信号
   - 趋势市 → 用 50 线 + 背离
4. **多时间框架验证**：日线出现底背离 + 周线 RSI 从 30 回升 → 信号更强。
5. **避免在重大事件前后依赖 RSI**：如财报、美联储决议，情绪驱动会扭曲技术信号。

---

## 九、一句话总结

> **RSI 不是买卖开关，而是市场“情绪温度计”——它告诉你市场是否过热或过冷，但最终决策，必须结合“天气”（趋势）和“地形”（价格结构）。**
