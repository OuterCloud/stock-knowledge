“**基本面调研框架**”是价值投资、行业研究、机构买方调研的核心。
下面我给你一个**全面、细致、可执行的股票基本面调研框架**，适用于港股、美股、A 股全市场。
我把它拆解成六个层级（从宏观 → 行业 → 公司 → 财务 → 管理层 → 风险），并且在每个环节里列出你能**实地查证或量化分析的指标**。

---

## 🧭 一级：宏观环境分析（Top-down 起点）

> 目标：确认市场处于哪一轮经济周期、政策周期。

| 维度                | 要点                                    | 数据来源                     |
| ------------------- | --------------------------------------- | ---------------------------- |
| **经济周期**        | GDP 增速、PMI、通胀、失业率、消费者信心 | Wind、TradingEconomics、CEIC |
| **货币环境**        | 利率、M2、央行政策、美元指数            | 各国央行、FRED               |
| **财政政策**        | 政府支出、减税政策、基建投资            | 财政部公告、新闻稿           |
| **国际形势**        | 地缘风险、全球贸易流、主要商品价格      | IMF、Bloomberg、Reuters      |
| **汇率 & 资本流动** | 对进出口型企业尤为重要                  | 香港金管局、SAFE、FRED       |

👉 **输出结论**：

> 当前宏观环境对该行业/公司的利润增长是“顺风”还是“逆风”。

---

## 🏭 二级：行业研究分析（确定竞争地形）

> 目标：确认行业成长性、竞争格局和壁垒。

| 维度                     | 要点                                                                        | 数据获取方式                   |
| ------------------------ | --------------------------------------------------------------------------- | ------------------------------ |
| **行业生命周期**         | 属于导入期/成长期/成熟期/衰退期？                                           | 行业报告（券商研报、咨询机构） |
| **市场规模与增速**       | TAM、SAM、SOM、增速 CAGR                                                    | iResearch、Frost & Sullivan    |
| **竞争格局（五力模型）** | 1. 新进入者威胁 2. 替代品威胁 3. 买方议价力 4. 供应商议价力 5. 现有竞争强度 | 同行业财报、新闻舆情           |
| **行业壁垒**             | 技术、品牌、资金、监管、渠道                                                | 专利库、资质要求               |
| **盈利模式**             | 主要盈利来源（销售、服务、订阅、金融利差等）                                | 公司年报、招股书               |
| **政策监管**             | 税收政策、许可证要求、反垄断政策                                            | 政府公报、行业协会             |
| **上下游生态**           | 上游（原材料/供应商）、下游（客户/渠道）依赖度                              | 财报披露、供应链地图           |

👉 **输出结论**：

> 行业是否具备高成长性？竞争格局是否健康？有无政策风险？

---

## 🧩 三级：公司核心面分析（护城河+竞争优势）

> 目标：判断公司在行业中的地位与可持续性。

| 模块           | 核心指标                                       | 调研要点                                   |
| -------------- | ---------------------------------------------- | ------------------------------------------ |
| **商业模式**   | 收入来源、客户结构、成本结构                   | 谁付钱？为什么？可复制吗？                 |
| **核心竞争力** | 技术壁垒、品牌力、网络效应、规模效应、渠道优势 | 举例分析：苹果的生态锁定、贵州茅台的品牌力 |
| **市场份额**   | 行业内排名、CR3/CR5 集中度                     | 行业报告、招股书                           |
| **产品分析**   | 产品结构、生命周期、创新能力                   | 产品矩阵、差异化                           |
| **用户粘性**   | 客户留存率、复购率、付费率                     | 年报指标或 App 数据                        |
| **国际化程度** | 海外收入占比、汇率敏感度                       | 财报分部披露                               |

👉 **输出结论**：

> 公司是否具备长期护城河？核心竞争力是否可持续？

---

## 💰 四级：财务质量分析（企业生命体征）

> 目标：判断公司真实盈利能力、成长性和财务安全性。

| 模块                  | 核心指标                             | 解读方式             |
| --------------------- | ------------------------------------ | -------------------- |
| **盈利能力**          | 毛利率、净利率、ROE、ROIC            | 与行业均值对比       |
| **成长能力**          | 营收增长率、EPS 增长率、现金流增长率 | 连续 5 年数据趋势    |
| **偿债能力**          | 资产负债率、利息保障倍数、流动比率   | 判断是否激进         |
| **现金流**            | 经营现金流/净利润比值、自由现金流    | 经营质量高低         |
| **资本支出（CAPEX）** | 投入产出比、资本开支周期             | 是否过度扩张         |
| **股东回报**          | 分红率、回购记录                     | 是否重视股东价值     |
| **财务舞弊信号**      | 应收账款/收入比、异常毛利率          | 参考 Beneish M-score |

👉 **输出结论**：

> 盈利结构是否健康？财务有无异常？现金流是否真实？

---

## 👥 五级：管理层与治理结构

> 目标：评估管理层诚信、能力、激励机制。

| 维度               | 要点                           | 调查方式             |
| ------------------ | ------------------------------ | -------------------- |
| **核心人物背景**   | 创始人/CEO 经历、教育、声誉    | 官网、媒体、LinkedIn |
| **管理团队稳定性** | 高管流动率                     | 年报变更公告         |
| **股权结构**       | 控股股东、实际控制人           | Wind、招股书         |
| **激励机制**       | 股权激励计划、期权授予         | 公司公告             |
| **公司治理质量**   | 独立董事比例、关联交易、审计师 | 年报审计意见         |
| **文化与愿景**     | 管理层言行一致性               | 投资者交流会、采访   |

👉 **输出结论**：

> 管理层是否值得信赖？治理结构是否透明？

---

## ⚠️ 六级：风险与估值分析

> 目标：识别不确定性、估算安全边际。

| 模块             | 内容                                 | 工具                |
| ---------------- | ------------------------------------ | ------------------- |
| **主要风险**     | 政策、行业、汇率、技术替代、关键人物 | 场景分析            |
| **估值方法**     | DCF、PE、PB、EV/EBITDA、PEG          | Excel、雪球、Finbox |
| **历史估值区间** | 与自身、同行对比                     | 图表分析            |
| **安全边际**     | 当前价格与合理估值差距               | 估值敏感度分析      |

👉 **输出结论**：

> 当前价格是否有吸引力？风险收益比是否合理？

---

## 🧾 七级：调研执行方法（实践部分）

| 手段              | 工具                           | 说明                     |
| ----------------- | ------------------------------ | ------------------------ |
| **财报拆读**      | 10-K / 20-F / 年报 / 中报      | 从 MD&A 和财务附注入手   |
| **券商研报**      | Wind、雪球、公众号             | 提炼核心假设，不盲从结论 |
| **舆情与资讯**    | Google News、微博、Reddit      | 观察市场情绪             |
| **实地/渠道调研** | 线下门店、App 数据、供应链跟踪 | 一手信息最有价值         |
| **数据分析**      | Python + Pandas / Excel        | 自建模型跟踪关键指标     |

3. 竞争分析
4. 财务表现
5. 估值与风险
6. 投资逻辑与结论（你的投资假设）
